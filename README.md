# Survey Analysis Workspace

Веб-приложение для анализа результатов количественных опросов: topline, crosstabs, heatmaps, корреляции, driver analysis и первичная обработка открытых ответов.

## Что умеет

- Загружает `.xlsx`, `.xls`, `.csv`.
- Автоматически распознаёт типы вопросов: single choice, multiple choice, шкала/rating, открытый текст, «Другое».
- Даёт проверить и исправить словарь вопросов.
- Строит topline по всем вопросам.
- Строит crosstabs: категория × категория, шкала × сегмент.
- Строит тепловые карты.
- Считает корреляции и простую driver analysis.
- Показывает открытые ответы, частотность слов и простую кодировку тем по ключевым словам.
- Экспортирует результаты в Excel.

## Как запустить через сайт без установки программы пользователями

Самый простой путь — Streamlit Community Cloud.

1. Создайте аккаунт на GitHub: https://github.com/
2. Создайте новый репозиторий, например `survey-analysis-workspace`.
3. Загрузите в репозиторий все файлы из этой папки:
   - `streamlit_app.py`
   - `requirements.txt`
   - `.streamlit/config.toml`
   - `README.md`
4. Откройте https://share.streamlit.io/
5. Войдите через GitHub.
6. Нажмите **Create app**.
7. Выберите репозиторий, ветку `main`, файл запуска `streamlit_app.py`.
8. Нажмите **Deploy**.
9. После запуска получите ссылку вида `https://ваше-имя.streamlit.app/`.
10. Отправьте эту ссылку коллегам.

## Важное про конфиденциальность

Если опрос содержит реальные данные пользователей или клиентов, лучше:

- делать GitHub-репозиторий приватным;
- в Streamlit включить приватный доступ;
- приглашать пользователей по email;
- не выкладывать сами Excel-файлы в репозиторий.

Файл опроса должен загружаться через интерфейс приложения, а не храниться в GitHub.

## Как запускать локально, если понадобится

Если всё-таки нужно запустить на своём компьютере:

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

После этого приложение откроется в браузере.

## Рекомендуемый порядок работы в приложении

1. Загрузить файл.
2. Перейти в «Словарь вопросов» и проверить типы вопросов.
3. Посмотреть «Качество данных».
4. Открыть «Topline».
5. Строить нужные «Crosstabs».
6. Смотреть «Heatmaps».
7. Проверить «Корреляции и драйверы».
8. Разобрать «Открытые ответы».
9. Скачать Excel-экспорт.
